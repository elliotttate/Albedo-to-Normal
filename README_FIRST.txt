================================================================
ALBEDO TO NORMAL MAP CONVERTER - QUICK START
================================================================

You're seeing this because you need to install Depth Anything V3!

QUICK FIX (Windows):
--------------------
1. Run this command:
   install_depth_anything.bat

2. Then run:
   run.bat


QUICK FIX (Linux/Mac):
----------------------
1. Run these commands:
   chmod +x install_depth_anything.sh
   ./install_depth_anything.sh

2. Then run:
   ./run.sh


WHAT DOES THIS DO?
------------------
This will:
- Clone the Depth Anything V3 repository (~200MB)
- Install the package and its dependencies
- Verify everything is working

It takes about 5-10 minutes depending on your internet speed.


ALTERNATIVE: Full Setup
-----------------------
If you haven't installed ANY dependencies yet, run:

Windows:    setup.bat
Linux/Mac:  ./setup.sh


VERIFY INSTALLATION
-------------------
At any time, check if everything is installed:
   python check_install.py


NEED HELP?
----------
See these files for more info:
- INSTALL.md    : Detailed installation instructions
- QUICKSTART.md : Quick start guide
- README.md     : Full documentation


COMMON ISSUES
-------------
Q: "git is not recognized"
A: Install Git from https://git-scm.com/downloads

Q: "python is not recognized"
A: Install Python 3.10+ from https://www.python.org/downloads/

Q: Still having issues?
A: Read INSTALL.md for detailed troubleshooting
